Capítulo 1

É possível adicionar no arquivo css uma regra ao colocar um @, como, por exemplo @charset "UTF-8";, para que o arquivo externo css também tenha a mesma compatibilidade de acentuação que o UTF-8 do arquivo html ao qual ele referencia. Existem diversas outras regras possíveis. Também é possível adicionar mais de um link de arquivo css em uma única página, e os outros estilos desse segundo link serão aplicados só às páginas que terão este link também (caso apenas algumas sejam diferentes das demais páginas). Em um único arquivo html, é possível adicionar um CSS externo, um CSS interno e também um CSS inline, todos em um mesmo arquivo HTML.
