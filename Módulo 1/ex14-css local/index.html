Capítulo 1

Para fazer o css local, basta criar uma tag <style> </style> abaixo da tag <title> </title>. Dentro da tag style, usa-se o seletor desejado (sem colchetes angulares), seguido de colchetes. Fica desta forma: 


body{
    background-color: lightsteelblue;
    font-family: Arial, Helvetica, sans-serif;
    font-size: 20px; 
}
h1{
    color: darkblue;
    background-color: red;
}
h2{
    color: darkred;
}
p{
    text-align: justify;
}

Capítulo 1.1

Esta técnica tem duas desvantagens: primeiramente, o estilo ocupa muitas linhas de código, e fica um código visualmente muito grande para uma página com conteúdo pequeno, e isso não é o mais agradável de se ver; além disso, caso seja necessário fazer esse tipo de estilos em dezenas de outras páginas, vai ser necessário copiar todo o código css e colar em todas as dezenas de arquivos html diferentes, o que torna igualmente inviável a manutenção de todos os arquivos html caso seja necessário alterar algum elemento CSS e este esteja local/interno.
