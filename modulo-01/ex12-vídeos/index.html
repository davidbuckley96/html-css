Sobre Vídeos

Um bom site para vídeos e fotos sem direitos autorais é o PEXELS.COM

Um EXCELENTE programa transcoder de formatos de vídeo é o HANDBRAKE

Para vídeos, existem os seguintes formatos que são compatíveis com navegadores: MP4 (.mp4 e m4v), WEBM (.webm) e OGG (.ogv {com v, em vez de g})

É importante ter vários formatos de arquivo na pasta de mídia do site, pois pode ser que o usuário visite o site em um navegador incompatível com alguns formatos de vídeo que você tenha

Inserindo vídeos hospedados localmente

Este vídeo está hospedado em meu próprio servidor

Neste caso, apenas um arquivo será executado, o que não resolve o problema de incompatibilidade com outros formatos de arquivos por cada navegador. O código para isso é este:

<video src="midia/Meu-Video.mp4" height="350" controls></video>

Para resolver o problema, basta criar uma tag <video> </video>, como fizemos com os vários formatos de áudio.

        
<video poster = "imagens/thumb.jpg" height = "400" width = "350" controls>
<source src="midia/meu-video.mp4" type="video/mp4">
<source src="midia/Meu-Video.m4v" type="video/mp4">
<source src="midia/video-ficticio.webm" type="video/webm">
<source src="midia/video-ficticio.ogv" type="video/ogg">
<p>Seu navegador não tem compatibilidade com a reprodução deste vídeo</p>
</video>

OBS: Se a thumb tiver dimensões diferentes da do vídeo, mas você na tag <video> adicionar ambos height e width, as novas dimensões irão ser aplicadas em ambos imagem e vídeo! (passei meia hora tentando resolver isso com a iolanda...)

Incorporando vídeos do YouTube

Basta ir ao vídeo desejado, selecionar "compartilhar", "incorporar" e copiar o código HTML. Após copiar, basta colar como ele está, e pronto. Não é preciso estar dentro de tags de vídeo, nem nada. É possível mudar as dimensões. Para fazer a inserção de vídeos do Vimeo, o processo é o mesmo, e no vimeo não há sugestão de outros vídeos ao pausar. No Youtube, os vídeos são livres para todos verem, e no Vimeo você pode colocar apenas para pagantes terem acessos ao conteúdo, por exemplo.
