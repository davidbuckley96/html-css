Links para download

  • Baixar o livro em pdf

    • Para pdf, basta adicionar um target = "_blank" para que o link abra em uma nova aba e fique mais cômodo para o usuário.

  • Baixar o arquivo .rar

    • Basta adicionar o local do arquivo, bem como "download = 'nome do arquivo'", mais "type = 'media type específico na lista do site iana.org'". Em alguns navegadores, basta colocar o nome do arquivo e já é feito o download.
