Paleta de cores

Usando o Adobe Color

Acessar o color.adobe.com e ele já de cara dá uma sugestão de paleta de cores. Assinar o adobe ajuda podendo salvar as paletas de cores, mas é possível utilizar sem salvar. É possível ver as paletas de cores desejadas e conseguir o código RGB das mesmas e ir experimentando ao vivo.

Ao clicar em "extrair tema", conseguimos importar um arquivo de imagem que tenhamos, e ele já dá as paletas de cores da imagem. Ao clicar novamente em "disco de cores", a paleta que foi gerada do arquivo importado já estará lá com os códigos

Ao clicar em "explorar", e restringir a exibição a "temas de cores", surge então uma lista com paletas de cores muito utilizadas para que possa escolher

Usando o Paletton

Acessar o paletton.com e ele já começa com a paleta de cores também. O interessante desse site é que ao clicar em "examples", ele aparece com uma sugestão de design de site, para ver como essas cores combinam juntas em um layout de uma página web.

Para usar esse recurso, entretanto, parece que existe um bug. É preciso primeiro acessar o facebook do Paletton (fb.com/paletton), e entrar no site pelo link deles na página. O link é o mesmo, mas o preview só funciona se for desta maneira.

Uma boa proposta é utilizar o Adobe Color para conseguir uma ótima paleta de cores, e depois ir ao Paletton para ver o layout do site de exemplo, para ver se as cores juntas fazem sentido.

Usando o Coolors

Acessar o coolors.co (não é .com), e para esse site basta apertar a barra de espaços para dar novas paletas de cores. Ao gostar de uma cor, podemos travar essa cor, e ao apertar a barra de espaço irá surgir novas cores que combinem com essa primeira cor que foi salva, e é possível travar várias cores, até o máximo das 5 cores sugeridas.
