Título

É necessário adicionar uma regra com o código 


@font-face {
    font-family: 'Raleway';
    src: url('fontes/Raleway-VariableFont_wght.ttf');
}
body {
    font-family: 'Raleway', sans-serif;
}
para que a fonte baixada seja adicionada corretamente.

Usa-se um nome qualquer para nomear a fonte ('Raleway', nesse caso), depois coloca-se a url com o nome do arquivo e a extensão correta (atenção ao nível das pastas onde essas fontes se encontram), e depois basta, no estilo, referenciar a fonte com o nome dado

As extensões dos arquivos de fontes podem ser truetype font (.ttf), opentype font (.otf), embedded-opentype, truetype-aat (Apple Advanced Typography), e .svg
