Em CSS as classes são referenciadas com um ponto ( . ) e os id são referenciados com hashtag ( # ). Já as pseudoclasses, essas são referenciadas com dois pontos ( : )

As pseudoclasses têm a ver com o estado do elemento (se está ativo, marcado, vazio, habilitado...)

Hover

O Hover é uma pseudoclasse que torna-se ativa ao passar o mouse em cima do elemento em questão. É possível, por exemplo, mudar a cor dos quadros ao passar o mouse em cima devido ao hover com background-color

Para aplicar a pseudoclasse hover em um elemento, usa-se a seguinte semântica:

div:hover {background-color: yellow; }

Elementos Filhos ( > )

No caso da div 4, existe uma div como as demais, mas que dentro dela há um texto escondido que só é revelado ao passar o mouse em cima. Para fazer esse exemplo, foi criada dentro da div 4 uma nova div chamada "escondido" <div class-"escondido">.

Nesta div escondido, foi feito dentro um texto sem parágrafo (textos dentro de divs aparecem na página html sem precisar estar em um parágrafo) que dizia "Passe o mouse aqui". Este texto foi marcado com um span, e dentro da div escondido também foi feito um parágrafo escrito "texto escondido!"

No arquivo CSS, para esconder o texto secreto, usou-se a propriedade display: none no .escondido > p

Isso quer dizer que o parágrafo p que está dentro da classe .escondido, ou seja, o p filho de .escondido, não terá display algum.

No arquivo CSS, para que o texto escondido aparecesse ao colocar o mouse em cima, usou-se: div.hover > p {display:block}, para então o display deixar de ser none e passar a ser visível com o block.

Entretanto, o texto surgia e o texto "passe o mouse aqui" continuava visível. Para resolver este problema, foi utilizado o span marcado com a classe .span1, onde esta passava a ter seu display:none. Como o código lê na ordem em que é escrito, primeiro ele revela o texto do .span1, depois o hover revela o texto do .escondido, ao mesmo tempo em que some com o display do .span1. Caso a ordem fosse contrária, e o span1 estivesse antes no código, a div 4 ficaria vazia, com o texto escondido e o display do "passe o mouse aqui" como none.
