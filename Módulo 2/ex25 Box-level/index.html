Exemplo de caixa box-level

Parágrafos também são exemplos de box-level, mas os links são exemplos de caixas inline-level. Vamos ver como tudo isso funciona.

OBS.: apesar de o <a> ser uma caixa inline-level, ela também pode ter border, margin, padding e outline definidos! Neste caso, foi dada uma borda amarela com padding de 1px só para ilustrar que é possível fazer isso em todas as boxes.

Neste caso, a caixa de conteúdo deixou de ocupar toda a viewport pois o width foi manualmente alterado para 73px. a caixa de conteúdo é a cor salmão

A border é a caixa verde, e tem uma característica de border-width 10px; border-color: green; border-style: solid;, para ficar bem visível

o padding adiciona um espaçamento entre o texto e a linha verde da border, mas não aumenta o tamanho da caixa de conteúdo, como pode parecer. O motivo disso é que não é possível adicionar cor ao padding, ele apenas afasta a borda do conteúdo

O margin adicionou um espaçamento no h1 para cima, baixo e esquerda (direita não foi necessário pois já está longe o suficiente da borda da direita

Caso queira centralizar no meio, NÃO DEVE USAR MARGIN À ESQUERDA!, basta usar o seguinte código: margin: auto. Este código centraliza a margem para o meio da tela, independentemente do tamanho da viewport do usuário

O outline é o pontilhado azul em volta da border, e é possível distanciá-la da border ao usar o código outline-offset: 50px;, com os pixels desejados.


Simplificando os códigos

SHORTHAND DE BORDER:

Para border, deve ser nesta ordem: largura -> estilo -> cor, ou seja, ficaria:

border: 10px solid black;


SHORTHAND DE PADDING:

Para padding, deve ser nesta ordem: topo -> direita -> baixo -> esquerda, como em um relógio, ou seja, ficaria:

padding: 10px 5px 10px 5px;

OBS.: SE AS DISTÂNCIAS DE PADDING FOREM TODAS IGUAIS (10px, 10px, 10px e 10px, por exemplo), PODEMOS APENAS DIZER: padding: 10px;

OBS.2: SE USAR APENAS DOIS VALORES PARA O PADDING POR EXEMPLO, padding: 10px 20px;, ISSO FARÁ COM QUE O PADDING DE CIMA E BAIXO SEJA 10px, E O DA DIREITA E ESQUERDA SEJA DE 20px.



SHORTHAND DE MARGIN

Para margin, deve ser nesta ordem: topo -> direita -> baixo -> esquerda, igual ao padding, lembrando que os valores podem ser substituidos por "auto"



SHORTHAND DE OUTLINE

Para outline, usa-se nesta ordem: largura -> estilo -> cor. Desta maneira, ficaria assim:

outline: 5px dashed salmon;
