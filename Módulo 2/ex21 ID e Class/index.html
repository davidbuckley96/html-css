Identificadores (id="principal" h1#principal)

Este título é um <h1>, mas, caso eu queira ter um estilo diferenciado que o do "Primeiro título de nível 1" e do "Segundo título de nível 1", é preciso adicionar um seletor para o mesmo. Para isso, adiciona-se na tag: <h1 id = "principal">

Agora, basta ir no código CSS e referenciar o identificador utilizado para dar características diferentes. Para referenciar um identificador, utiliza-se a seguinte semântica: h1#exemplo {}

As características de <h1> serão as mesmas para todas as tags <h1>, mas propriedades diferentes das dos restantes <h1> serão aplicadas apenas ao identificador.

Por exemplo, se tanto <h1> quanto <h1 id="exemplo"> tiverem a mesma cor de texto, então não é preciso declarar a cor para <h1 id="exemplo">, mas, se o tamanho de texto for diferente, o tamanho declarado em <h1 id="exemplo"> irá sobrescrever o tamanho de texto dos demais <h1>, mas mantendo todas as demais propriedades que não forem diferentes.

OBS.: APENAS PODE-SE TER 1 IDENTIFICADOR POR DOCUMENTO HTML! POR MAIS QUE FUNCIONE TER MAIS DE 1, ESTÁ ERRADO! PARA MAIS DE UM MARCADOR, USA-SE CLASS.

Primeiro título de nível 1 (class="primeiro" .primeiro)

Neste caso, foi utilizado um <h1 id="primeiro">, e no arquivo CSS foi declarado que a cor de .primeiro {} seria amarela. Neste caso, todos os seletores genéricos .primeiro terão a mesma configuração de estilo.

Segundo título de nível 1 (class="segundo" .segundo)

Neste caso, foi utilizado um <h1 id="segundo">, e no arquivo CSS foi declarado que a cor de .segundo {} seria azul. Neste caso, todos os seletores genéricos .segundo terão a mesma configuração de estilo.

Primeiro título de nível 2 (class=primeiro" h2.primeiro)

Neste exemplo, a classe usada foi a mesma, mas, caso eu queira que o h2 com a classe de 'primeiro' tenha alguma coisa diferente, basta no CSS declarar h2.primeiro, em vez de apenas .primeiro

Segundo título de nível 2 (class="segundo" h2.segundo)

Neste exemplo, a classe usada foi a mesma, mas, caso eu queira que o h2 com a classe de 'segundo' tenha alguma coisa diferente, basta no CSS declarar h2.segundo, em vez de apenas .segundo

Neste exemplo, temos um texto dentro de um parágrafo comum, mas eu desejo que esta frase específica fique vermelha. Para isso, usa-se uma tag nova chamada <span> </span> para marcar o trecho da frase específica, e então, na tag de abertura adicionar a classe que terá a proprpiedade de cor diferente. A sintaxe é a seguinte: <span class = "span1">esta frase específica fique vermelha</span>

OBS.: É possível adicionar mais de uma classe a um mesmo elemento html. Para isso, basta adicionar a seguinte sintaxe: 


    <h2 class = "exemplo1 exemplo2">Título com duas classes<h2>
Também é possível adicionar um id e uma classe juntos, mas as propriedades do id sobrepõem-se às da classe, caso sejam conflitantes.
