Gradientes/Degradê

Para fazer gradientes, não utiliza-se a propriedade background-color, mas sim background-image, mesmo que o fundo não seja um arquivo de imagem em si. O código fica assim: 

        
        body {
        background-image:
        linear-gradient(to [direção desejada] right, #2B9EB3, #8340A0);
        }

No gradiente linear, indica-se primeiro a direção, e em seguida quantas cores queremos.

Caso a direção seja para cima, escrevemos linear-gradient(to top), mas ficará zebrada, listrada, e não é o que queremos. Para resolver isso, usamos uma configuração global (o atalho de configuração global é * { } )
