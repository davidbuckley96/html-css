O html5, por padrão, torna os links não visitados azuis, e os visitados ficam roxos. É possível sobrepor essas definições em CSS, como já foi feito neste exemplo.

Neste caso, o código é o seguinte:

ul > li > a {
    text-decoration: none;
    color: #d6d6d2
    }

Para que o texto fique sublinhado ao passar o mouse, usa-se o seguinte código:

ul > li> a:hover {
    text-decoration: underline;
    }

Para que o texto fique diferente ao permanecer com o mouse clicado no mesmo, usa-se o seguinte código: 

ul > li > a:active {
    color:yellow;
    }

Pseudoclasses ( : )

  • Visited

    Serve para configurar o estilo de links visitados

  • Hover

    Serve para configurar o estilo de elementos ao passar o mouse em cima dos mesmos

  • Active

    Serve para configurar o estilo de links ao permanecer com o mouse clicando nos mesmos

Pseudoelementos ( : : )

Pseudoelementos têm impacto no conteúdo dos elementos

Por exemplo, se desejar adicionar após cada link um texto dizendo que é um link, é possível fazer o mesmo com um pseudoelemento. Para isso, usa-se o elemento + ::

a::after {
    content: ' -> Este é um link!';
    color:aqua
    }

Nota-se que o texto do pseudoelemento também faz parte do link, e, se for clicado, também redicionará para o site em questão!
